### CDN : Content delevery(Distribution) network

- 인터넷 콘텐츠를 고속 전송하는 지리적으로 분산된 서버 집단
- HTML / JS / CSS / 이미지 등 인터넷 콘텐츠를 로드하는데 필요한 자산을 신속하게 전송 가능
- 그냥 외부 서버(컴퓨터)에서 가져와서 사용할게
  - jsDelivr : 쓰면, 그냥 링크 따와서 해주면 알아서 해줌
